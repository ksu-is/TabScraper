# TabScraper Roadmap

## Version 1 (MVP)
- Extract tab titles
- Store tabs in a Python list
- Print results to console

## Version 2
- Capture URLs + titles
- Save data to a file (CSV or JSON)
- Improve error handling

## Version 3
- Build browser extension (Edge/Chrome)
- Add UI for viewing tabs
- Enable real-time tab tracking

## Future Features
- Search tabs
- Export data
- AI-powered tab organization
